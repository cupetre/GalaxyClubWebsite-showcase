# GalaxyClub — Billiard Club Reservation Website

GalaxyClub is a full-stack reservation platform built for a billiard club / coffee house environment, designed to provide a modern user experience for customers and a clean technical foundation for future business use.

> Note: this public repository contains the frontend showcase version of the project. Backend implementation and production configuration are kept private.

## Overview

The goal of this project was to build a modern reservation website where users can explore the venue, view key information, and make reservations through a clean and intuitive interface.

The project focuses on:
- polished landing-page design
- structured Angular component architecture
- reservation-focused user flow
- scalable frontend organization for future expansion

## Features

- Landing page with brand-focused presentation
- About section with visual content
- Reservation section with interactive date and time selection
- Contact section
- Responsive layout
- Component-based Angular structure

## Tech Stack

- Angular
- TypeScript
- HTML
- CSS
- REST API integration on the frontend side
- Docker / deployment-ready project structure in the full private version

## Project Structure

The frontend was built with Angular using reusable components and a structured separation of layout, content sections, and reservation logic.

Main areas include:
- Navbar
- Hero section
- About section
- Reservation interface
- Contact section
- Footer

## Reservation Concept

The full project was designed around a reservation workflow for billiard tables, with time-based availability handling and backend support for booking validation.

In the full private implementation, the frontend communicates with a backend service responsible for reservation processing and availability management.
